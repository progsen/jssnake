## Setup

- maak een `git repo` aan voor dit vak `m2flexjs`
- clone die `git repo` naar je laptop
- open de `directory` die je krijgt (`m2flexjs`) in visual studio code

- maak 2 files aan:

```
- index.html
- app.js
```

 ![](img/htmlcanvas.PNG)
